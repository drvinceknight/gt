Question 1¶

For each of the following pairs of matrices, for the corresponding normal form game use support enumeration to output all Nash equilibria as a list.

a. $ A = \begin{pmatrix}1 & 2 \\ 2 & 1\end{pmatrix} \qquad B = \begin{pmatrix}3 & 2 \\ 2 & 5\end{pmatrix} $

Available marks: 2

In [1]:
import nashpy as nash
import numpy as np

### BEGIN SOLUTION
A = np.array([[1, 2], [2, 1]])
B = np.array([[3, 2], [2, 5]])
game = nash.Game(A, B)
list(game.support_enumeration())
### END SOLUTION
Out[1]:
[(array([0.75, 0.25]), array([0.5, 0.5]))]

Create a variable strategy_space which is a tuple containing all strategies of the repeated game.
